## 项目介绍

一个基于 Electron + Vue 3 开发的现代化 API 测试工具，类似于 Postman 的功能实现。

### 功能特点

- 🚀 支持多种 HTTP 请求方法（GET, POST, PUT, DELETE 等）
- 📝 支持多种请求体格式
  - form-data
  - x-www-form-urlencoded
  - raw (JSON, Text, JavaScript, HTML, XML)
  - binary
  - GraphQL
- 🔍 内置 cURL 命令解析器
- 📦 请求历史记录管理
- 🎨 支持多种主题（浅色、深色、蓝色、绿色）
- 🌍 国际化支持（中文、英文）
- 💾 支持响应数据下载
- 🖥 跨平台支持 (macOS, Windows, Linux)

### 技术栈

- Electron
- Vue 3
- TypeScript
- Vite
- Ant Design Vue
- Pinia
- Axios

### 开发环境要求

- Node.js >= 16
- yarn >= 1.22.0

### 安装和运行

1. 克隆项目
```bash
git clone [项目地址]
cd lbs-test
```

2. 安装依赖
```bash
yarn install
```

3. 开发模式运行
```bash
yarn electron:dev
```

4. 构建应用
```bash
yarn electron:build
```
